# Code of Conduct

## Preamble

We want `goo-core` to be a welcoming and constructive place for everyone. This Code of Conduct outlines expectations for behavior in the project, including issues, pull requests, and discussions.

## Expected behavior

All community members are expected to:

- Be respectful and constructive.
- Assume good intent.
- Provide clear and actionable feedback.
- Help keep discussions focused on improving the project.

## Unacceptable behavior

Examples of behavior that are not allowed include:

- Harassment, hate speech, or discrimination.
- Threats, intimidation, or targeted abuse.
- Posting private or sensitive information.
- Disruption of discussions or repeated off-topic behavior.

## Reporting

If you believe someone is violating this Code of Conduct, please report it to the project maintainers via GitHub (issue or direct message).

## Enforcement

Project maintainers may take action deemed necessary and appropriate, including warning the offender, removing comments/content, temporarily limiting participation, or banning repeat offenders.

Actions will be taken in a fair and timely manner based on the information available.
